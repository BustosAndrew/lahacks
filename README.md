# Chef.ai
AI recipe generator web app all made in Python using the [reflex](https://reflex.dev/) framework and Gemini API!

## Installation
```
git clone https://github.com/BustosAndrew/lahacks.git
cd lahacks
pip install -r requirements.txt
reflex run
```
